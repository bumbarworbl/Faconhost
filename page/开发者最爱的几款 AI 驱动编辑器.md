在探索开发者对于 AI 驱动编辑器的偏好时，我首先问了自己以下问题：

## 数据源与背景

早期用户的数据通常能够揭示工具的创新点。然而，许多早期采用者使用的工具并未成为主流，这往往是因为现有供应商会根据客户需求调整其工具，防止新竞争对手抢占市场份额。

在目前的市场上，**Visual Studio**、**Visual Studio Code** 和 **JetBrains IDE** 是主流的集成开发环境（IDE）。而它们的竞争对手则是近年来涌现的新兴 IDE 初创公司。

## 开发者最喜欢的 AI 驱动编辑器

根据开发者的反馈，以下是目前最受欢迎的AI驱动编辑器：

1. **Cursor**：一款以 AI 为核心的代码编辑器，正迅速普及。
2. **Visual Studio Code（结合 GitHub Copilot）**：大多数使用 Microsoft 免费 IDE 的开发者表示，他们在此 IDE 中使用了 GitHub Copilot。
3. **JetBrains IDE（结合 GitHub Copilot）**：JetBrains 提供针对特定语言的 IDE，如 IntelliJ、PyCharm、PHPStorm 和 RubyMind。
4. **Windsurf 编辑器**：由 Codeium 最新发布的“代理 IDE”，迅速吸引了大量用户。
5. **Zed**：专为与 AI 协作设计的下一代代码编辑器。
6. **Neovim 与 Copilot**：基于 Vim 的文本编辑器，支持多种扩展，包括 GitHub Copilot。
7. **Supermaven**：虽非 IDE，但可作为 AI 完成插件运行于 VS Code 或 JetBrains IDE 中。
8. **Aider**：基于终端的结对编程助手，需与 IDE 并行使用。
9. **Cline**：一个集成到 IDE 中的自主编码代理。

这些工具都利用了 AI 模型来生成代码，尽管这些操作需要成本，但一些工具提供了免费的有限版本，而对于专业开发者来说，付费版本的价格也相当合理。

## Cursor：开发者的新宠

Cursor 是一款成立于 2022 年的 AI IDE 初创公司，于 2023 年 3 月发布了其首个版本。在 2024 年 7 月，随着对 Sonnet 3.5 模型的支持，Cursor 的受欢迎程度显著提升。

### Cursor 的优势
- **上下文保存与规则管理**：Cursor 能够保存上下文并管理规则，这使得开发者能够更高效地工作。
- **速度**：与其他工具相比，Cursor 的速度更快。
- **Composer**：这是一个类似 AI 代理的功能，能够跨多个文件工作，根据提示实现功能。

开发者 Roman Tsegelskyi 分享了 Cursor 成为他最爱 IDE 的原因：

> “在尝试了竞争对手的产品后，Cursor 仍然是我的最爱。它规则和保存上下文的能力、更快的速度以及 Composer 功能，都让我觉得它能带来更好的结果！”

此外，Cursor 的代码生成能力、聊天窗口、多行编辑功能以及对不同模型的切换，都使其成为开发者的首选。

## Windsurf：AI 协作的新星

Windsurf 是 Codeium 开发的一款新发布的 AI 代码编辑器，专注于改善与 GenAI 的协作流程。

### Windsurf 的亮点
- **Cascade**：类似于 Cursor 的 Compose，Cascade 是一个代理聊天机器人，支持跨多个文件协作，并可运行终端命令。
- **Cascade 的工作流程**：Cascade 的索引和代码上下文处理优于 Cursor 的 Compose，尤其是在快速发展的代码库中。

产品经理 Amul Badjatya 对比了 Cursor 和 Windsurf 的使用体验：

> “Windsurf 的 Cascade 工作流程优于 Cursor 的 Compose，特别是在代码库快速发展的情况下。Cascade 还能运行终端命令，这一点非常重要。”

## Zed：免费 AI 驱动的选择

Zed 于 2023 年 3 月公开发布，是一款由 14 名开发者构建的 IDE，并且是唯一为注册用户提供免费、无限制 AI 完成的工具。

### Zed 的特点
- **Vim 模式**：适合喜欢 Vim 用户界面的开发者。
- **快速响应**：编辑器和 LLM 的响应速度非常快。
- **键盘映射**：支持配置为使用习惯的 IDE 键盘映射。
- **多种模型**：与多种 LLM 完美集成。

开发者 Siddhart Jha 分享了他对 Zed 的喜爱：

> “向 AI 上下文窗口提供特定文件、选择模型、使用 Copilot/Supermaven 的补全功能都非常简单。我喜欢 AI 功能不夺人眼目，在需要时才会显示。”

## Visual Studio Code 与 JetBrains IDE 的现状

许多开发者仍然在使用 Visual Studio Code 或 JetBrains IDE，但主要依赖于代码自动完成功能。相比之下，Cursor 和 Windsurf 的多文件编辑和代理功能更受青睐。

JetBrains AI 的优势在于其价格与 GitHub Copilot 相同，但许多开发者反馈其代码提取效率不如竞争对手。

## 未来的 AI 驱动 IDE 革命

在过去 18 个月中，许多 AI 驱动 IDE 已推出并获得了发展势头，预计未来会有更多创新。
